About DevOps

DevOps is a set of practices that combines software development (Dev) and IT operations (Ops). It aims to shorten the systems development life cycle and provide continuous delivery with high software quality. Popular DevOps tools include Jenkins, Docker, Kubernetes, Ansible, Terraform, and many more.
